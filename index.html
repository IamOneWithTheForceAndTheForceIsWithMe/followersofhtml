Here are the list of resources and their description.

  1. overAPI.com - Collecting All Cheat Sheets (HTML, CSS, JS, and other as Java, php, Python, Ruby, ...).
  2. Cheat sheet of HTML - here one can download a pdf or png version of a compact cheat sheet.
  3. How to Code in HTML5 and CSS3 - a book that helps people to start making websites.
  4. HTML validator - allows you to check html code whether it contains syntax errors.
  5. Google HTML/CSS Style Guide - describes how to make your code well formatted.
  6. Mozilla developer network - advanced manual to web technologies.
  7. Can I use...? - allows you to check whether a tag is supported into different browsers.
  8. FreeCodeCamp - HTML, CSS, and JavaScript trainer.
  1. CSS validator - allows you to check css code whether it contains syntax errors.
  2. Block element modifier - describes an approach of naming in HTML and CSS
  3. CSS diner - CSS selectors trainer.
  4. Flexbox Froggy - CSS flexbox trainer.
  5. Flexbox Defense - CSS flexbox trainer.
  6. Grid Garden - CSS grid trainer.
  7. Specificity Calculator - a visual way to understand CSS specificity.
  8. PURE CSS GAME - NO JS - contains examples of games written on pure CSS code.
  9. Learn CSS Layout - this site teaches the CSS fundamentals that are used in any website's layout.
  10. CSS-Tricks - teaches you life hacks about how to manage problems with CSS.
  11. Autoprefixer CSS online - autoprefixer is a PostCSS plugin which parse your CSS and add vendor prefixes.
  12. Grid by example - this site is a collection of examples, video and other information to help you learn CSS Grid Layout. Developed and maintained by Rachel Andrew.
  13. CSS Tools: Reset CSS - allows to reset default browser's style.
  14. Normalize.css - makes browsers render all elements more consistently and in line with modern standards. It precisely targets only the styles that need normalizing.
  1. Object playground - about OOP into JS.
  1. Softserve IT academy, moodle - to view PowerPoint presentations, schedule , and auditoriums availability.
  2. moqups.com - allows to make up a web page and share with others. Free version does not have a lot of options, but it is easy way to fashion a page and share it with your teacher/colleagues.
  3. ninjamock.com - alternative to moqups.com.
  4. Stepik.org - fascinating website with lots of free courses about different field of IT and natural science.
  5. ColorMania - ColorMania is an advanced color picker utility. Main feature - Onscreen Eyedropper (Color grabber).
  6. Programming mentor - the web page of our admired mentor ;).
